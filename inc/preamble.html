This page tracks the deployment of RDAP among top-level domains. It is updated once per day (last update: {{ TODAY | escape }}).

Description of columns:

  • TLD - the TLD name. Hover to see the A-label version of IDN TLDs.
  • Type - the "type" of the TLD. This comes from the IANA root zone database.
  • Domains - the approximate number of second-level domains under the TLD (taken from various public sources)
  • Port 43 - whether a whois server is registered for the TLD at IANA. Some TLDs have whois servers even if IANA has no entry
  • RDAP - whether there is an RDAP Base URL in the Bootstrap Service Registry
  • Added - the date that an RDAP Base URL was first observed for this TLD in the Bootstrap Registry
  • HTTPS - whether at least one URL in the registry for this TLD has the https:// scheme
  • DNSSEC - whether at least one URL in the registry for this TLD has a host which is covered by a valid RRSIG record
  • DANE - whether at least one URL in the registry for this TLD has a corresponding TLSA record (this record is not validated at the moment)
